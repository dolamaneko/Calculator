# Simple JS Calculator CLI

A simple command-line calculator built with Node.js.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Easy to use from the terminal

## Requirements

- Node.js installed

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/simple-js-calculator.git
cd simple-js-calculator
```

Install dependencies:

```bash
npm install
```

## Usage

Run directly with Node:

```bash
node index.js 5 + 3
```

Or install locally as a CLI:

```bash
npm link
calc 10 * 4
```

## Available Operators

- `+` for addition
- `-` for subtraction
- `*` for multiplication
- `/` for division

## Example

```bash
calc 8 / 2
```

Output:

```bash
Result: 4
```
